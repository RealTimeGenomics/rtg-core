Usage: Phylogeny [OPTION]... -o DIR -d DIR

Required flags: 
  -o, --output=DIR      directory for output
  -d, --subject=DIR     directory containing formatted subject dataset

Optional flags: 
  -h, --help            print help on command-line flag usage
  -P, --progress        report progress
  -r, --repeat-freq=INT maximum repeat frequency (Default is 1000)
  -S, --step=INT        step size (Default is window size)
  -W, --word=INT        word size (Default is 25 for slimn and 9 for all other
                        modes)
